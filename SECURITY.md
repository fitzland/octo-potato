# Security Policy

## Supported Versions

Use the latest version of the site/docs.

## Reporting a Vulnerability

Please report security vulnerabilities via GitHub's [Private vulnerability reporting](https://github.com/fitzland/octo-potato/security/advisories/new) feature.

Alternatively, email me at f1tzl4nd@gmail.com with details. I'll acknowledge receipt within 96 hours and aim to resolve critical issues quickly.
